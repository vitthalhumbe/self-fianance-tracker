# Finance App

A self-hosted personal finance application to track expenses, income, and pending receivables.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Lucide Icons
- **Backend**: Node.js, Express
- **Database**: SQLite (better-sqlite3)

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Initialize the database (first run only):
   ```bash
   node server.db.js
   ```

## Usage

start both backend (port 4000) and frontend (port 4001):
```bash
npm start
```
